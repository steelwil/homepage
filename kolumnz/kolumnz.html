Kolumnz Game

This is a clone of the old classic game Kolumnz.
Stackpack is a similar Tetris clone.

Whats new

2005-03-12: Update, prettier graphics, Open sourced, License under the GPL.

2001-05-04: Fixed minor bug. Sometimes the ini file did not load properly.

Object of the game

This game is similar to Tetris, except here you align 3 or more blocks on any plane.

Screen Shot

Features

  • Single or Double players.
  • Works on Windoze 95, 98, NT, W2K, XP (MicrosoftÂ® Internet Explorer 4.0 or greater must be installed)
  • Customizable games.
  • Customizable playing keys.
  • Customize your playing grid
  • Choose pieces containing up to 15 colours.
  • Stretchable screen.
  • No splash screens, irritation screens, adverts or reminders.
  • Stand alone, no need for Direct-X or any other system files.
  • Reasonably small footprint.
  • Does not write to the registry.
  • And the best of all it's free! Licensed under the GPL.
